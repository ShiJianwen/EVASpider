Vicky 532 1 天前 回答

Java只支持单继承，而且要求每个继承类的构造函数必须调用父类构造函数，也就是说每个类都需要调用其一个父类的构造函数（只有一个）。
所以你的问题是随便你如何定义构造函数，但是必须调用父类也就是A.B.C的构造函数，也就是有一个String参数的构造函数，写法就是：

public Test(String str){
    super(str);// 显示，且在第一行调用父类的构造函数
    // 其他代码
}

当然上面只是一个例子，你也可以定义一个无参构造函数，如：

public Test(){
    super(null);// 显示，且在第一行调用父类的构造函数，参数为null
    // 其他代码
}
